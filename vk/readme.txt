Importing VKontakte (vk.com) posts into your WordPress site.

== Description ==

With this plugin you can easily import any [**VKontakte**](https://vk.com/) posts into your **WordPress** site.

= Working with plugin =
* After installation you should go to the plugin's page in the ***Tools*** section.
* Enter *vk.com* page address to load data.
* Choose posts range to import.
* You can also set the target records type (post, page) and their initial state (draft, publish).
* Finally, click the ***Import*** button and wait until the end of operation.

== Installation ==

Install the plugin through the **WordPress** installer or copy the plugin's files into your site's */wp-content/plugins/* folder and then activate it via the control panel.
